Health companion System

A responsive front-end web application that simulates a Smart Hospital Management System.  
This project allows patients to enter medical details, select symptoms, book appointments, and communicate with doctors. Doctors can manage appointments and interact with patients through chat and video features.

---

Project Description

The Smart Hospital System is a single-page web application built using HTML, CSS, and JavaScript.  
It demonstrates a basic hospital workflow including patient registration, doctor registration, symptom-based doctor matching, appointment booking, notifications, and chat functionality.

This project is created for educational purposes, mini-project submission, hackathons, and front-end development practice.



Features

Patient Module
- Role-based login selection
- Enter personal details
- Enter basic health details
- Select symptoms
- Find doctors based on symptoms
- Book appointments
- Chat with doctor
- Send files in chat
- Video call access

 Doctor Module
- Doctor profile entry
- Clinic details entry
- Appointment notification dashboard
- Accept or reject appointments
- Chat with patients
- Video call feature

---

 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Maps Embed
- Web Camera API (getUserMedia)

---

How It Works

- The application uses a slider-based navigation system.
- Doctors are matched based on selected symptoms.
- Appointment data is stored temporarily using JavaScript arrays.
- Chat messages are dynamically created using DOM manipulation.
- Video functionality uses browser camera permissions.

---

How to Run the Project

1. Download or clone this repository.
2. Open the HTML file in any modern web browser (Google Chrome recommended).
3. Allow camera permissions to use the video call feature.
4. Select your role (Patient or Doctor) and explore the system.

No installation or backend setup is required
